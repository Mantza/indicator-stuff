Methodology

AI notes are identified by joining the notes file with userEnrollment on the note author's participant ID, filtering for users whose enrollmentState is apiEarnedIn. A note is counted as "helpful" if its currentStatus in the noteStatusHistory file is CURRENTLY_RATED_HELPFUL. The date used is the note's creation date. Data source: X Community Notes public data.
